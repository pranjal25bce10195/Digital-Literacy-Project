This project is created as part of the Digital Literacy course. It covers important topics such as digital awareness, building student digital portfolio,coding and collaboration platforms, email etiquette, and cybercrime safety.

Created a digital literacy awareness infographic using Canva, covering key topics like safe internet practices and digital tools.

Developed professional profiles on GitHub, LinkedIn, and Kaggle to build an online presence.

### Task 3 – Platforms
Completed a coding challenge on HackerRank and created a Digital Literacy Awareness quiz using Google Forms.

### Task 4 – Email Etiquette
Drafted two professional emails and created a social media Do’s and Don’ts checklist.

### Task 5 – Cybercrime Awareness
Prepared a case study on UPI fraud and created a prevention checklist.
